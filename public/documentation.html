Instructions

Through this app, you may:

  • Use the top-left navigation menu to view other pages
  • Leverage the home page's filter-based system by complete the form and view results â€” clickable listings that direct users to farmers market websites
  • View results through the home page's interactive map

The application currently relies solely on the POST method, which processes form data, queries a back-end database based on user input, and sends matching results to the front-end, which are displayed in both the "Results Table" and the interactive map visible on the home page. For more information, please consult the Developer Manual corresponding to this application's Github repository.
